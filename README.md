# VS Code Web Interface Engine

A high-fidelity replica of the Visual Studio Code interface, focused on complex Web App Architecture and scalable CSS engineering.

## Tech Stack
- **HTML5:** Semantic markup for IDE structures.
- **CSS3:** Advanced Grid, Flexbox, and Variables.
- **SVG:** Icon system based on VS Code Codicons.

## How to Run
1. **Clone the repository** to your local machine:
   ```bash
   git clone [https://github.com/Kimnkitae/VS-Code-Web-Interface-Engine](https://github.com/Kimnkitae/VS-Code-Web-Interface-Engine)
2. cd VS-Code-Web-Interface-Engine
3. Launch the application:

Locate the index.html file in the root folder.

Open index.html by double-clicking it or dragging it into any modern web browser (Chrome, Firefox, or Edge).

Tip: If you are using VS Code, use the "Live Server" extension for the best development experience.
